About Me

Welcome to my website! My name is Colin Vincent and I am a 4th year Combined Electrical and Computer Engineering Student at Northeastern University. I am passionate about electronics and embedded systems and I am currently searching for my 3rd co-op placement.

Lumenpulse Embedded Systems Co-op

At Lumenpulse I worked on several microcontroller projects as part of the firmware team. I also spent time working with Altium for schematic design and board layout. My main projects were Lumenfacade Rev 6, CBOX and Lumentone.

I designed the PCB layout for the newest revision of the Lumenfacade White HO and RO boards through several iterations. This included several hardware changes in the schematics, and significant changes to the board layout.

The CBOX directs communication over DMX and RDM. I worked both of these protocols to fix existing issues with interference coming into the receiver.

The picture to the left is from an R&D project to incorporate an OLED into a Lumentone light controller. Using the STM32F030R8 development board I prototyped a set up to control lights over DMX with an interactive menu displayed on the OLED screen.

I also worked with an existing test jig to accommodate a new control card. This required several hardware changes on the PCB layout and new firmware for the STM microcontroller.

Draper Electrical Engineering Co-op

At Draper Laboratory I worked with the Sensors and Imaging Systems group. I was responsible for the electrical system in a pod test set up. I also worked with two other co-ops on Wireless Sensor Network Research.

I spent a lot of time working with OrCAD capture to review PCB schematics and order parts for the manufacturer. I also used OrCAD to compile wiring diagrams for the pod project.

Researching wireless sensor networks led our team to develop a project focused on camera based networks. After spending a significant amount of time reading existing journal publications we chose to focus on surveillance using a network of cameras.

We developed a project for a proof of concept for 3-D reconstruction using a network of cameras. This was done with Raspberry Pi’s with camera modules. My contribution included the hardware selection and the shell scripting to communicate with the Pi’s and transfer images to the host computer. This resulted in a conference paper presented in May 2016. The paper can be found here: http://ieeexplore.ieee.org/document/7568905/?reload=true “

Naval Undersea Warfare Center Student Technical Intern

At NUWC I worked with REMUS autonomous underwater vehicles. I was part of the payload integration team and spent a lot of time in the lab assisting with vehicle testing. I worked with OrCAD to make wiring diagrams for some of the hardware we were working to integrate in the REMUS.

Projects Misc

Android App

I built this app to concatenate three .wav files and play the result given a specific code. There are 19 built in wav files and the app can create a output given any combination. The generated sound is used to trigger underwater liftbags.

LED Cube

Using an Arduino Pro I connected several WS2812 addressable LEDs to the serial output. Then using a Bluetooth module I connected the Arduino to my Raspberry Pi using python. I then pulled NOAA weather information so I could adjust the color and intensity of the LEDs depending on the current weather in Boston. I covered the hardware with a cube I made out of Plexiglas that I frosted with sandpaper. Some of the code is up on GitHub.
